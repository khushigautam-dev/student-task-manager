#  Student Task Manager

##  Overview
Student Task Manager is a simple Java console-based application designed to help students organize and manage their daily academic tasks efficiently.

---

##  Problem Statement
Students often forget important assignments, deadlines, and tasks due to poor task management.

---

##  Solution
This project provides a simple system where users can:
- Add tasks
- View all tasks
- Delete completed tasks

---

##  Features
-  Add new tasks  
-  View all tasks  
-  Delete tasks  
-  User-friendly console interface  

---

##  Technologies Used
- Java  
- ArrayList (Collection Framework)  
- OOP Concepts  

---
